# Sobre o projeto

O **Aticurando** é um sistema desenvolvido como Projeto Integrador (PI) da Fatec, com o objetivo de auxiliar no gerenciamento de pacientes, atendimentos e registros relacionados ao processo de interação entre cuidadores e idosos.

A aplicação foi construída utilizando **ASP.NET Core (C#)** no backend e **HTML/CSS** no frontend, com banco de dados **MySQL** administrado via **phpMyAdmin / XAMPP**.

O sistema permite cadastro, listagem e controle das principais entidades do processo, oferecendo uma base sólida para futuras melhorias e expansão do projeto.

# Como executar o projeto

## Backend
Pré-requisitos:  
- .NET 8 SDK instalado  
- Visual Studio ou VS Code

```bash
# clonar repositório
git clone https://github.com/matos795/AticurandoPI.git

# entrar na pasta do projeto
cd AticurandoPI

# restaurar dependências e executar
dotnet run
```
